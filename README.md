Лабораторная работа №5: Регулярные выражения в Java
Описание

Работа посвящена изучению и практическому применению регулярных выражений (Regex). Регулярные выражения представляют собой мощный инструмент для поиска, проверки (валидации) и модификации текстовой информации на основе заданных шаблонов.
Основные инструменты

В языке Java работа с Regex строится на двух основных классах пакета java.util.regex:

&nbsp;&nbsp;&nbsp;&nbsp;Pattern: Скомпилированное представление регулярного выражения.

&nbsp;&nbsp;&nbsp;&nbsp;Matcher: Движок, который выполняет операции сопоставления с текстом, используя объект Pattern.

Синтаксис и квантификаторы

В проекте использованы следующие элементы шаблонов:

&nbsp;&nbsp;&nbsp;&nbsp;`\d — любая цифра.`

&nbsp;&nbsp;&nbsp;&nbsp;`\w — любая буква или цифра.`

&nbsp;&nbsp;&nbsp;&nbsp;`\b — граница слова.`

&nbsp;&nbsp;&nbsp;&nbsp;`{n, m} — повторение от n до m раз.`

&nbsp;&nbsp;&nbsp;&nbsp;`[A-Z] — набор символов (в данном случае — заглавные латинские буквы).`

&nbsp;&nbsp;&nbsp;&nbsp;`^ и $ — начало и конец строки соответственно.`

Содержание заданий
1. Поиск чисел в тексте (NumberFinder)

&nbsp;&nbsp;&nbsp;&nbsp;Программа извлекает все числовые значения (включая целые и дробные) из произвольного текста.


2. Валидация пароля

&nbsp;&nbsp;&nbsp;&nbsp;Реализована проверка пароля на соответствие строгим критериям безопасности:

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Длина от 8 до 16 символов. Только латинские буквы и цифры. Обязательное наличие минимум одной заглавной буквы и одной цифры.


3. Исправление регистра (CapitalAfterLower)

&nbsp;&nbsp;&nbsp;&nbsp;Программа находит ошибки набора текста, где заглавная буква идет сразу за строчной.


4. Проверка IP-адреса

&nbsp;&nbsp;&nbsp;&nbsp;Шаблон для проверки корректности IPv4-адреса. 4 блока чисел, разделенных точками. Каждое число в диапазоне 0–255.

5. Поиск слов по первой букве

&nbsp;&nbsp;&nbsp;&nbsp;Гибкий поиск слов, начинающихся с определенного символа, игнорируя регистр.
